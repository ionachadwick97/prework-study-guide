HTML

  • HTML is the building blocks of the webpage and gives it structure.
  • Head elements contain metadata for the browser to read.
  • Body elements contain visible information for users to read.
  • Meta and header elements aren't visitble to users on the webpage other than the title element which shows in the tab bar.

CSS

  • CSS is the style and flare of the webpage.
  • It's linked to the HTML code in the header element.
  • Margin is how much space is around the outside of an element.
  • Padding is how much space we want around the content inside an element.

Git

  • git status: checks what branch we are currently in.
  • git checkout -b branch-name: creates new branch and swithces to it.

JavaScript

  • JavaScript is what makes the webpage interactive.
  • A variable is a named container that allows us to store data in our code.
  • Control flow is the order in which a computer executes code in a script.
  • console.log describes action to be taken.
  • Funtions describe an action to be taken, but will only execute when called.
